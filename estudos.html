Estudando posicionamentos dentro da pagina.

CAMPO DE LOGIN:  

 * RELATIVE: quando aplicado teremos 5 propriedades para trabalhar= *top *right *bottom *left *z-index, Como: "Position: relative; left: 100px; top: 80px;" Quando você usa este elemento o local do seu elemento continua, só que vazio, ou seja um espaço.

 * ABSOLUTE: também libera as cinco propriedades porem de maneira diferente, e como se ele críace uma página em cima da outra, ele transfere o elemento para essa página eliminando o espaço vazio movimentando todos os outros elementos, Sé o elemento estiver sujeito a uma tag pai e essa tag estiver pedindo= "position: relative;" o elemento cera relativo a tag pai como: "Main { position: relative; }"

 * FLEX: mais facilidades= nos permite os elementos dentro de caixas, controle em uma dimensão (horizontal ou vertical), alinhamento, direcionamento, ordenamento e tamanho. *Display: flex;= alinha na horizontal, *flex-direction: column;= volta para vertical, justify-content: spece-between; ou space-around; ou space-evenly;= coloca um espaço entre os elementos, justify-content: center;= os elementos juntos mas ao centro, obs: quando você modificar o tamanho da sua janela os elementos vão estar sempre se ajustando, sempre se mantendo no centro, como no exemplo a baixo.

 * GRID: posicionamento horizontal e vertical ao mesmo tempo, pode ser flexível ou fixo, cria espaços para os elementos filhos habilitarem.

 * GRID OU FLEXBOX: podemos usar grid e flex juntos, obs: só que não no mesmo elemento.
