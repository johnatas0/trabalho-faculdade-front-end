Atividade de Programação Front-End

Objetivo da Atividade

Com base no nosso material de estudo e nas pesquisas complementares, você deverá:

  • Criar um website responsivo de tema livre (portfólio pessoal, blog, loja virtual, etc.) usando HTML5, CSS3 e JavaScript. O site deve ter pelo menos 4 páginas interligadas.
  • Aplicar conceitos de Layout Responsivo, Menu de Navegação Funcional, Formulário de Contato com Validação via JavaScript, Efeitos Interativos e SEO Básico.
    •

    Requisitos Específicos

    Seu website deve incluir:

  • Layout responsivo que se adapte a diferentes tamanhos de tela.
  • Menu de navegação funcional.
  • Formulário de contato com validação via JavaScript.
  • Uso de pelo menos dois efeitos interativos com JavaScript/jQuery.
  • Aplicação de SEO básico no código.
    •

    Documentação

    Crie um relatório técnico explicando:

    Escolhas de design e arquitetura do site.

  • Tecnologias utilizadas no desenvolvimento do site.
  • Desafios enfrentados durante o desenvolvimento.
  • Capturas de tela do resultado final.
  • Explicações sobre como o projeto aplica os conceitos estudados.
    •

    Entrega da Atividade

    Hospede seu site em plataformas como GitHub Pages, Netlify ou Vercel (opcional) e inclua o link no seu relatório, junto com o código-fonte compactado em formato ZIP.

    A atividade deve ser enviada exclusivamente através do Template de entrega da atividade MAPA, disponível em "Arquivos >> Material da Disciplina (Studeo)".
